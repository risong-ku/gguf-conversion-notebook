# GGUF Conversion Notebook

Convert Hugging Face models to GGUF format for local inference (LM Studio / llama.cpp).

## Features

- Transformers → GGUF conversion
- Quantization support (Q2_K ～ Q8_0)
- Works with LM Studio

## Usage

Open the notebook in Google Colab and run all cells.

- Quantized GGUF model
